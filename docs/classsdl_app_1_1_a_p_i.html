Detailed Description

Used to define, implement, and load a Lua API for the sdlApp::State class.

In addition to API handles for the sdlApp::State class, this class implements various globals, such as the entire SDL_Scancode enum, global events, and related functions. The object 'Application' is used to call methods of sdlApp::State in Lua scripts, e.g. Application:GetFramerate();
